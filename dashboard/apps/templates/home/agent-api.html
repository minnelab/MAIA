MAIA Agent API & MCP Server
Active

Connect an AI agent or any MCP-compatible client to manage users and projects through natural-language commands.

POST Generic REST Agent
/maia/agent/chat/

Send a plain-text command and receive the agent's response. Supports multi-turn conversation via the history field.

POST /maia/agent/chat/
X-Agent-Token: <AGENT_API_TOKEN>
Content-Type: application/json

{
  "message": "List all users",
  "history": []
}

Response: {"response": "...", "history": [...]}

POST Mattermost Integration
/maia/agent/mattermost/

Configure as a Mattermost Outgoing Webhook or Slash Command integration. The agent posts results back to the channel.

Mattermost setup:

  1. Go to Integrations → Outgoing Webhooks
  2. Set Callback URL to
    https://<your-host>/maia/agent/mattermost/
  3. Set the token to match AGENT_API_TOKEN
  4. Optionally set a trigger word, e.g. maia

Or use a Slash Command with the same URL and token.

POST HTTP MCP Server
/maia/agent/mcp/

Stateless JSON-RPC 2.0 endpoint implementing the Model Context Protocol. Connect any MCP-aware client or agent framework.

POST /maia/agent/mcp/
X-Agent-Token: <AGENT_API_TOKEN>
Content-Type: application/json

{"jsonrpc":"2.0","id":1,
 "method":"tools/list","params":{}}

Methods: initialize, tools/list, tools/call

Standalone MCP Server (stdio transport)

The file mcp_server/maia_mcp_server.py is a full MCP server that connects to this dashboard via its REST API. Use it with Claude Desktop, Cursor, Windsurf, or any other MCP client.

Claude Desktop — claude_desktop_config.json:

{
  "mcpServers": {
    "maia-admin": {
      "command": "python",
      "args": ["/path/to/MAIA/mcp_server/maia_mcp_server.py"],
      "env": {
        "MAIA_API_URL": "https://{{ hostname }}/",
        "MAIA_API_TOKEN": "<admin-DRF-token>"
      }
    }
  }
}

To obtain a DRF token for an admin user run (inside the dashboard container): 

python manage.py shell -c "
from django.contrib.auth.models import User
from rest_framework.authtoken.models import Token
u = User.objects.get(username='<admin-username>')
token, _ = Token.objects.get_or_create(user=u)
print(token.key)"
