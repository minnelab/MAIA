Integration Reference

Send a natural-language command; supports multi-turn conversation via the history field.

POST /maia/agent/chat/
X-Agent-Token: <AGENT_API_TOKEN>
Content-Type: application/json

{
  "message": "List all users",
  "history": []
}

→ {"response": "...", "history": [...]}

Configure a Mattermost Outgoing Webhook or Slash Command.

  1. Go to Integrations → Outgoing Webhooks
  2. Callback URL: https://{{ hostname }}/maia/agent/mattermost/
  3. Token: matches AGENT_API_TOKEN
  4. Trigger word (optional): e.g. maia

Users then type maia list all projects in any channel.

Stateless JSON-RPC 2.0 endpoint. Connect any MCP-aware agent or framework.

POST /maia/agent/mcp/
X-Agent-Token: <AGENT_API_TOKEN>
Content-Type: application/json

# Handshake
{"jsonrpc":"2.0","id":1,"method":"initialize","params":{}}

# List tools
{"jsonrpc":"2.0","id":2,"method":"tools/list","params":{}}

# Call a tool
{"jsonrpc":"2.0","id":3,"method":"tools/call",
 "params":{"name":"list_users","arguments":{}}}

Use mcp_server/maia_mcp_server.py with Claude Desktop, Cursor, or any MCP client.

# claude_desktop_config.json
{
  "mcpServers": {
    "maia-admin": {
      "command": "python",
      "args": ["/path/to/MAIA/mcp_server/maia_mcp_server.py"],
      "env": {
        "MAIA_API_URL": "https://{{ hostname }}/",
        "MAIA_API_TOKEN": "<admin-DRF-token>"
      }
    }
  }
}

Get a DRF token (inside the dashboard container):

python manage.py shell -c "
from django.contrib.auth.models import User
from rest_framework.authtoken.models import Token
u = User.objects.get(username='<admin-username>')
t, _ = Token.objects.get_or_create(user=u)
print(t.key)"
